# HuiDesktop

谢谢！HuiDesktop NextGen给很多人带来了很多快乐和安慰（1.0版本蓝奏云下载次数接近3w次），但是由于一些不可接受的限制性（Chromium、WebGL……），是时候和它说再见了。

此仓库会一直保留，供后来者查看其代码。谢谢各位！
